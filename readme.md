### Run Project

### Install Prereqs

    ```
    sudo apt install -y \
        git \
        ruby \
        ruby-dev \
        build-essential
    ```

### Install Jekyll

    ```
    sudo gem install jekyll bundler
    ```

### Install Gems

    ```
    bundle install
    ```

### Build
    ```
    bundle exec jekyll serve
    ```
